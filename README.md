Image-Based Time of Day Classification using MATLAB

Overview

This project classifies images as Day or Night and estimates the corresponding time period (Morning, Noon, Evening, or Night) using traditional image processing techniques without machine learning.

Features

Manual RGB to Grayscale conversion
Manual RGB to HSV conversion
Brightness analysis
Sky color detection
Color temperature estimation
Artificial light detection
Rule-based classification system
Feature visualization and reporting

Technologies

MATLAB
Image Processing
Feature Extraction
Rule-Based Decision Making

Day/Night classification
