# Image-Based Time of Day Classification using MATLAB

## Overview
This project classifies images into Day or Night and estimates the time period (Morning, Noon, Evening, or Night) using traditional image processing techniques without machine learning.

## Features
- Manual RGB to Grayscale conversion  
- Manual RGB to HSV conversion  
- Brightness analysis  
- Sky color detection  
- Color temperature estimation  
- Artificial light detection  
- Rule-based classification system  
- Feature extraction and visualization  

## Technologies
- MATLAB  
- Image Processing Toolbox  
- Feature Extraction  
- Rule-Based Decision Making  

## Output
The system provides:
- Day/Night classification  
- Estimated time period  
- Extracted feature report  
- Visualized feature charts  

## Notes
No machine learning models were used; the classification is fully rule-based using handcrafted features.
